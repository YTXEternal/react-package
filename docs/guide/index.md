# 简介

UX Table React 是一个基于 React 开发的高性能表格组件库。它专为大数据量和复杂交互场景设计，内置了丰富的数据网格（DataGrid）功能，在保持轻量级的同时提供极致的渲染性能。

## 为什么选择它？
- ⚡️ **极致性能**：内置虚拟滚动引擎与 Web Worker 数据处理，支持无限滚动加载，即使面对十万级甚至百万级的数据也能保持丝滑流畅。
- 🛠 **功能丰富**：开箱即用的排序、内联编辑、固定列、拖拽列宽调整等核心表格功能。
- ✨ **Excel 级交互**：支持行列/多单元格选中、带蚂蚁线动画的剪贴板复制粘贴、以及基于撤销/重做栈的 `Ctrl+Z` / `Ctrl+Y` 操作。
- 🎨 **样式高度定制**：基于 UnoCSS 构建，提供灵活的样式定制和网格空位补齐（类似 Excel 画布）能力。
- 📦 **TypeScript**：全量 TypeScript 编写，杜绝 `any`，类型安全，提示友好。
